# 获取组件的属性

  • vm.$options.name 可以获取组件的 name 属性，用于业务逻辑
  • vm.$parent 获取父组件实例
  • vm.$el 获取当前组件 DOM
  • vm.$children 获取当前组件的所有子组件的 实例

总结，学到了什么？

  1. 封装的UI组件通信
  2. 通过当前组件获取相邻组件进行业务逻辑
