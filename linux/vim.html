# 基础使用vim

有三种模式

  1. 正常模式 vim 文件进入
  2. 编辑模式 i 进入
  3. 命令行模式 : 进入

在正常模式下

  • G 到最后一行
  • gg 到第一行
  • dd 删除当前行
  • u 撤销
  • ctrl + r / control + r 恢复
  • o 新增一行&进入编辑模式

命令行模式

  • :wq 保存退出
  • :q! 退出不保存
